We provide a lightweight standardized interface that exposes workflows through a unified JSON schema. This interface acts as a thin wrapper over the core workflow runner, mapping structured inputs to pipeline execution. It does not introduce new computational functionality; instead, it offers a consistent entry point for external systems to invoke existing workflows without modifying core code.

The interface is compatible with function-calling systems and provides a standardized mapping to existing pipelines. The interface package is distributed as a manifest, a schema definition, and usage instructions, and is included with this submission. No runtime extension mechanism, plugin registry, or dynamic loading is introduced; the interface package is a packaging and interface layer only.
